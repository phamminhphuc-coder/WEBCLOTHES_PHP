## 📖 About the Project

ClothesStore is a front-end e-commerce web application that allows users to explore clothing collections, filter products by category, and complete purchases through a smooth checkout flow. Designed with a clean UI and mobile-first approach, it delivers a seamless shopping experience without any dependencies.

---

## ✨ Features

### 🛍️ Product Catalog
- Browse products by category (Đồ nam, Đồ nữ)
- Product cards with image, name, price, and quick-add button

### 🛒 Shopping Cart
- Add / remove items from cart
- Quantity adjustment
- Real-time total price calculation

### 👤 User Account
- Login / Register forms
- User profile page with order history

### 💳 Checkout
- Multi-step checkout form
- Order summary before placing order
